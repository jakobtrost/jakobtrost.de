# Jakob Trost Website

Personal website for Jakob Trost, Web Developer.

## Tech

- Static HTML, CSS, and JavaScript
- No build step. Serve the folder with any static server or open `index.html` locally.
- Suited for [GitHub Pages](https://pages.github.com/) or any static hosting.

## Features

- Five color schemes (dark, vibrant, orange, cyan, aqua), randomly chosen on load
- Responsive layout, interactive card tilt and ripple effect
- Color scheme switcher and contact links

## Structure

```
├── index.html    # Main page
├── style.css     # Styles
├── script.js     # Behavior and color scheme logic
├── fonts/        # Clash Display
└── img/          # Favicons and assets
```
